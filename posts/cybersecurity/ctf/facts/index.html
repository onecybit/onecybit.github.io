PORT      STATE SERVICE VERSION

80/tcp    open  http    nginx 1.26.3
54321/tcp open  http    Golang net/http server (MinIO)

Port 80 redirects to http://facts.htb/. Add the host to /etc/hosts:

echo "10.129.39.130  facts.htb" | sudo tee -a /etc/hosts
